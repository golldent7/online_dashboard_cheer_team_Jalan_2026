# داشبورد دوري الهواة شجع فريقك 2026

## التشغيل المحلي
افتح ملف `index.html` مباشرة من المتصفح.

## النشر أونلاين
1. ارفع الملفات إلى GitHub Repository جديد.
2. من Settings > Pages اختر Branch: main و Folder: /root.
3. سيظهر رابط عام للداشبورد.

## المميزات
- ترتيب تلقائي حسب النقاط وكسر التعادل.
- إدخال النتائج والبطاقات.
- احتساب اللعب النظيف: صفراء = 1، حمراء غير مباشرة = 3، حمراء مباشرة = 4.
- حفظ تلقائي في المتصفح LocalStorage.
- تصدير واستيراد نسخة احتياطية JSON.
- طباعة الداشبورد أو حفظه PDF.

## ملاحظة مهمة
هذا الإصدار Static Web App لا يحتاج سيرفر. بيانات الإدخال تحفظ في جهاز الشخص الذي يدخل النتائج. إذا أردت إدخال جماعي من عدة أعضاء وتحديث مباشر للجميع، يلزم ربطه بقاعدة بيانات مثل Firebase أو Supabase.
